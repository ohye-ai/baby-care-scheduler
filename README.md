# 👶 宝宝照护排班表

一个帮助新生儿家庭科学安排照护轮换的免费工具。

## ✨ 特点

- 🆓 **完全免费** - 无需注册登录
- 🔒 **隐私安全** - 数据保存在本地浏览器
- 📱 **跨平台** - 支持电脑、手机、平板
- ⚡ **智能排班** - 自动生成公平的照护排班
- 🎨 **简洁易用** - 温馨友好的界面设计

## 🚀 快速开始

1. 在浏览器中打开网站
2. 添加家庭成员（爸爸、妈妈、外公外婆等）
3. 设置每个人的可用时间
4. 点击"自动生成排班表"
5. 可手动调整或导出分享

## 📋 功能介绍

### 成员管理
- 添加 2-6 位家庭成员
- 设置角色（爸爸/妈妈/外公/外婆等）
- 区分"上班"和"不上班"类型

### 智能默认规则
- **上班成员**：工作日只有晚上可带，周末全可带
- **不上班成员**：工作日全可带，周末不可带
- 可自定义修改任何时段

### 自动排班
- 智能分配照护时段
- 优先满足"偏好"时段
- 自动平衡每个人的工作量
- 支持锁定特定时段

### 导出分享
- 导出 CSV 表格
- 复制文本发送到家庭群
- 随时查看统计数据

## 💡 使用建议

### 浏览器选择
为了数据持久保存，建议：
- **iPhone/iPad**：使用 Safari 浏览器
- **Android**：使用 Chrome 浏览器
- **电脑**：使用 Chrome 或 Safari

⚠️ **重要提示**：
- 排班数据保存在当前浏览器本地
- 首次使用哪个浏览器，以后就用同一个浏览器
- 避免在微信内打开（数据会被锁定在微信中）

### 多设备使用
- 建议指定一台设备作为"排班管理设备"
- 或使用导出功能分享给家人
- 目前不支持自动跨设备同步

## 📱 跨平台支持

- ✅ Chrome（Windows/Mac/Android）
- ✅ Safari（Mac/iPhone/iPad）
- ✅ Firefox
- ✅ Edge
- ⚠️ 微信浏览器（可用但数据隔离）

## 🔧 技术栈

- 纯前端单页应用
- 无需后端服务器
- localStorage 本地存储
- 响应式设计
